The Orchestrator

Powered by Gemini 3 Pro. Acts as the project manager. It analyzes raw user input, determines the intent (e.g., "Create a script"), and delegates tasks to sub-agents.

2

Visuals as Code

Powered by Gemini 3 Flash. Instead of generating pixelated text in images, this agent writes Mermaid.js and HTML code to render crisp, editable diagrams.

3

Native Audio

Powered by Gemini 2.5 Native Audio. Transforms text scripts into human-quality speech with specific tonality (Formal vs. Casual) for auditory learning.

4

Generative Imagery

Powered by Gemini 2.5 Flash Image (Nano Banana). Creates consistent, vector-style illustrations to visualize step-by-step procedures.
